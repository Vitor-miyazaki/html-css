Link externo

Para colocar um link externo, utilizo a seguinte estrutura de tag's:
<a href="https://www.google.com.br/" target="_blank" rel="external" hreflang="pt-br"> Clique aqui para acessar o google </a>

→ Nofollow

Pelo que entendi, utilizando o parâmetro "nofollow", você basicamente diz ao google que não tem conhecimento desta página externa, ou seja, que você "não os segue". Isso é basicamente uma ferramenta que gera algum tipo de impacto para os mecênismos de buscas do google. Geralmente é utilizada para sites patrocinados. Exemplo:

<a href="https://www.magazineluiza.com.br/" target="_blank" rel"nofollow"> Clique aqui para acessar o site da Magazine Luiza </a>
