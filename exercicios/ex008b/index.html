Para explicar, o que houve anteriormente. Foi adicionado código da línguagem Python. Entretanto, para ficar com a estruturação ácima, foi utilizado duas tags.

A tag <code> serve para que todas as letras tenham o mesmo espaçamento, dessa forma facilitando a visualização de cada letra e número para o entendimento do código

A tag <pre> serve para que as frases, palavras, letras fiquem igual ao que postamos no Visual Code. Por exemplo, neste caso, no navegador o código ficou a esquerda e com as mesmas quebras de linha que tem no arquivo do VS code.

Um macete interessante que foi aprendido durante este exercício, é de colocar o parágrafo a esquerda ou a direita, de forma mais ágil. Quando clicamos em "Tab", o parágrafo vai para a direita. Quando clicamos em "shift" + "Tab" o parágrafo vai para a esquerda.

Significado das abreviações

Sempre que utilizar uma abreviação, podemos utilizar a tag <abbr>. Esta tag, faz com que ao passar o mouse por cima desta abreviação, apareça uma pequena legenda explicando o nome completo deste componente. Isso facilita tanto ao usuário, como os mecânimos de busca. Por exemplo, HTML e CSS.

Texto invertido / Texto invertido

A tag <bdo dir="rtl ou ltr"> serve justamente para dar uma ordem para as letras, sejam elas da direita para esquerda "rtl" ou da esquerda para a direita "ltr".
