Categorias de fontes

  1. font serif → Com serifa: Bom para títulos/destaques;
  2. font san-serif → Sem serifa: Bom para textos longos;
  3. font monoespaçada → Espaços iguais para cada glifo: Normalmente usada para representar códigos (fonte terminal ou console);
  4. font script → Letra de mão: Bom para títulos/destaques;
  5. font display → Fonte diferentes das anteriores, que não existe um padrão: Títulos/destaques.
