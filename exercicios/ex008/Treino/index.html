TESTE DE FORMATAÇÃO DE TEXTO

→ Texto em negrito

Texto em negrito utilizando a tag <b> (não semantico)

Texto em negrito utilizando a tag <strong> (semantico)

→ Texto em itálico

Texto em itálico representado pela tag <i> (não semantico)

Texto em itálico representado pela tag <em> (semantico)

→ Texto marcado

Para ter um texto marcado, utilizamos a tag <mark>. E para mudança da cor, precisamos definir o estilo, utilizando css.
→ Caso queira a mudança da cor, em apenas uma tag, então basta modificar o estilo naquela mesma tag utilizando a tag <mark style="background-color:lime;">
→ Caso queira a mudança da cor em todas as tag's, então é necessário configurar o estilo na tag <head>.

→ Texto inserido

A tag para o texto inserido é <ins> (semântica)
Também existe a tag <u> (não semântica)

→ Texto deletado

A tag que representa o texto deletado ou riscado é: <del>

→ Texto sobrescrito

Por exemplo: X20+3, foi utilizado a tag <sup> no número 20.

→ Texto subscrito

Por exemplo: H2O, foi utilizado a tag <sub> no dúmero 2.

→ Palavres grandes e pequenas

Para fazer uma palavra grande, podemos utilizar a tag <big> (não semântica)
Para fazer uma palavra pequena, então utilizamos a tag <small> (semântica)
