FORMATAÇÕES DE TEXTO

Negrito

→ Tag <b> (Sem semântica)
→ Tag <strong> (Com semântica)

Itálico

→ Tag <i> (Sem semântica)
→ Tag <em> (Com semântica)

Inserido/sublinhado

→ Tag <u> (Sem semântica)
→ Tag <ins> (Com semântica)

deletado

→ Tag <del> (Com semântica)

marcação

→ Tag <mark> (com a cor "aqua" definida para toda a página, dentro da tag <head> utilizando o CSS)
→ Tag <mark style="background-color: lime;"> (Com cor definida específicamente nessa frase)

Maior e menor

→ Tag <big> (sem semântica)
→ Tag <small> (com semântica)

Sobrescrito e subscrito

→ Tag <sup> / Exemplo: X2000+10
→ Tag <sub> / Exemplo: H2O e CO2
