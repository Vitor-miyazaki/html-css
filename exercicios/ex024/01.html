Usando a Label

Sendo que

  • autocomplete → Sugere um preenchimento automático do formulário, com base nos preenchimentos anteriores;
  • Action → Destino do envio/submit deste formulário. Este destino pode ser uma página de php, java ou outra linguagem de programação;
  • Label → Relaciona o texto (nome/sobrenome) com o input, ajudando as ferramentas de buscas (google por exemplo) e facilita no uso por celular;
