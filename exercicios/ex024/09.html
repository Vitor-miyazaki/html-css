Formulários







O problema dos formulários em HTML é a sua incopatibilidade com diferentes tipos de navegadores. A solução para tal problema é utilizar formulários com programação, por exemplo com Javascript.

Existem 2 tipos de metódos para formulários, sendo o GET e o POST. Devemos usar o POST para:

  • Formulários que ultrapassem o peso de 3000 bites, equivalente a 3000 carácteres. Formulários do tipo "file" por exemplo.

  • Segurança, afinal os dados pessoais do usuário não vão aparecer na URL, logo é gerado menos exposição. Neste caso, os formulários do tipo "password" são um exemplo. OBS: Por mais que tenha mais segurança, ainda assim devemos ter conhecimento em "https", pois só assim é possível garantir completa ocultação dos dados.
