TIPOS DE LISTAS

  1. Listas Ordenadas

    1. Lista que possui uma sequência
    2. Representada pela TAG <ol> combinada com a tag <li>
    3. Números, letras e algarísmos romanos (1,A,a,I,i). Utilizando a tag <ol type="1">
    4. A tag <ol type="A" start="5"> cria a lista a partir do 5° algorítmo, conforme o exemplo abaixo:
      1. Teste
      2. Teste
      3. Teste
    5. Para criação das listas podemos utilizar o comando (li*3). Esse comando indica que o sistema deve fazer 3 listas com a tag <li>. Isso ajuda na velocidade da criação.

  2. Listas Não Ordenadas

    • Lista que não possui uma sequência lógica
    • Representada pela TAG <ul> combinada com a TAG <li>
    • Disc, square, circle. Utilizando a TAG <ul type="circle">, conforme o exemplo abaixo:
      • Teste
      • Teste
      • Teste

  3. Listas de Definição

    1. TAG <dl> → Definition List. Tag que inicia e termina uma lista de definição.
    2. TAG <dt> → Definition Term. Tag que define o termo
    3. TAG <dd> → Definition Description. Tag que descreve o significado do termo
    HTML
    Línguagem de marcação para determinar o conteúdo da página
    CSS
    Línguagem de marcação que define o estilo
    JAVA
    Línguagem de programação que permite a interatividade do usuário com a página
