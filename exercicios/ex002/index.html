Revisão da aula 02

Aprendemos que os parágrafos podem ser representados pelas tags <p> e </p>, e que para fazer uma quebra de linha
como essa, precisamos utilizar as tags <br>. Um dos maiores erros de alguém que quer se tornar um profissional de programação, é abusar do recurso de quebra de linhas <br>, afinal podemos controlar os espaçamentos através do CSS que veremos futuramente.

Para utilização de emojis, precisamos acessar o site https://emojipedia.org/. A tag utilizada para a inserção do emoji é dada por &#x na sequência colocamos o código do emoji "codepoints".
✔ - O codepoints desse emoji era U+2714, mas utilizamos apenas "2714". Dessa forma a marcação foi criada dessa forma: &#x + 2714.
😊 - Codepoints U+1F60A
🔥 - Codepoints U+1F525

Existem tags específicas para diversos símbolos, segue alguns exemplos:
< > Menor que (Less than) e maior que (greater than)
® Marca registrada
™ Trademark
© Copyright
¥ - Yen
£ - Libra / pound
€ - Euro
¢ - Cents
δ e Δ - Delta
↑ Seta para cima
↓ Seta para baixo
← Seta para a esquerda
→ Seta para a direira
⇅ Seta para cima e para baixo
⇆ Seta para a esquerda e direita

Um site que contém esses símbolos é: https://www.w3schools.com/html/html_entities.asp

Caso queira utilizar um comentário no meu documento de html, sem que apareça na minha página, utilizo a tag <!--comentário-->.
