Como criar as tags básicas e para que servem

Dentro de uma Tag html contém outras tags, sendo elas "head" e "body".

Na tag "head" conseguimos dar instruções para a nossa página

- Alteração da línguagem da página para "pt-br"

- Conferir se o dimensionamento da tela está para tela cheia = viewport, na escala 1.0

- Colocar/editar o título da página

Já na tag "body" é onde desenvolvemos o conteúdo dessa página

- Função de título "h1"

- Função de parágrafo "p"

- Função de linha "hr"
